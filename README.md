# powerbi-reports
This repository showcases my Power BI Portfolio featuring end-to-end business intelligence solutions across multiple domains including:

Financial KPI Dashboard

Hotel Booking Analytics Dashboard

Insurance Portfolio Performance Dashboard

Marketing KPI Dashboard

Sales Target & Performance Dashboard

Each project demonstrates data modeling (star schema), Power Query transformations (ETL), advanced DAX calculations, KPI tracking, and interactive visual storytelling.

The dashboards are designed to convert raw business data into actionable insights, supporting strategic decision-making across finance, sales, marketing, insurance, and hospitality domains.
